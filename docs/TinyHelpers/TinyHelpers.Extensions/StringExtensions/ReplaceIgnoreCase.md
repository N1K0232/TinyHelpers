# StringExtensions.ReplaceIgnoreCase method

Returns a new string in which all occurrences of a specified string in the current instance are replaced with another specified string, ignoring the letter casing.

```csharp
public static string ReplaceIgnoreCase(this string input, string pattern, string replacement)
```

| parameter | description |
| --- | --- |
| input | The original string. |
| pattern | The string to be replaced, case insentive. |
| replacement | The string to be used to replace all occurrences of *pattern*. |

## Return Value

A string that is equivalent to the *input* string except that all instances of *pattern* are replaced with *replacement*.
