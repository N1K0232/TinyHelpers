# AuthenticatedParameterizedHttpClientHandler.SendAsync method

Calls the function to automatically add the Bearer token and then sends an HTTP request to the inner handler to send to the server as an asynchronous operation. If the response is 401 (Unauthorized), it will try to refresh the token using the handler specified in the [`AuthenticatedParameterizedHttpClientHandler`](../AuthenticatedParameterizedHttpClientHandler.md) constructor and try again.
